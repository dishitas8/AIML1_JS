var is used to declare variables.A variable is a container that stores some value (number, text, boolean, etc.) so that it can be used later in the program.

1️⃣ console console is an object provided by the browser. It represents the developer console where messages can be printed. 2️⃣ log log is a method (function) inside the console object. It means record or display a message.

Why we use typeof We use typeof to check what kind of value a variable contains. In JavaScript, a variable can store different types of data like: number string boolean object undefined typeof helps us identify which type it is.

In JavaScript, null represents an empty or non-existent value, but the typeof operator returns "object" due to a historical bug in the language.
